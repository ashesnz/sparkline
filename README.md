#Sparkline

An html version of a [Sparkline](http://en.wikipedia.org/wiki/Sparkline).


##Install
> bower install sparkline



##Usage

> sparkline( id, options )

See index.htm for example usage, but this,

    <div id="s1"></div>
    <script>
    sparkline( "s1",
             {
                "data": [10,15,7,100,100,100,94,23,77]
             });
    </script>

will get you going.
